# Geant4_NaI_project
Детектор NaI (Tl), расчет собственного фона детектора. Цилиндрический сцинтиллятор NaI диаметром 2.54 см и длиной 15 см помещен в алюминиевую оболочку с внешним диаметром 3.0 см и толщиной 1 мм для защиты от влаги. С одной из сторон кристалл просматривается через стеклянное окно толщиной 2 мм китайским ФЭУ N2014-25mm.
